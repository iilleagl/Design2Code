# 鼠标精度校准

用于评估点击准确度与反应时延的网页工具，建议定期测试以保持操作效率。

## 运行方式

直接用浏览器打开 `index.html`，或本地起一个静态服务：

```bash
# 在 aim-trainer 目录下
npx serve .
# 或
python3 -m http.server 8080
```

然后访问 http://localhost:8080 或 http://localhost:8080/index.html

## 测试模式

- **标准校准**：30 个校准点，不限时。用于基准测试与设备校验。
- **快速评估 · 常规**：30 秒内完成尽可能多的有效点击，校准点较大。
- **常规评估**：60 秒标准校准。推荐用于日常精度检查。
- **高精度校准**：45 秒小校准点。用于精细操作场景的评估。

## 指标说明

- 校准分：有效点击得分（含反应时延加权）。
- 有效 / 无效：点击到校准点记为有效，点空记为无效。
- 准确率：有效点击数 / 总点击数。
- 结果页显示：校准分、准确率、有效点击数、用时。
