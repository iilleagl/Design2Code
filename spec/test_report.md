二、 设计细节与规范
配色方案：

背景： #F5F7FA (浅灰背景)，卡片背景为纯白 #FFFFFF。

阴影： 卡片带有极细微的投影（box-shadow: 0 2px 8px rgba(0,0,0,0.05)）和圆角（通常为 4px 或 8px）。

状态色： 成功绿 (#52C41A)，危险红 (#F5222D)，进行中蓝 (#1890FF)，警告橙 (#FAAD14)。

字体： 系统默认无衬线字体（PingFang SC, Microsoft YaHei），字号层级清晰（标题 16-18px，正文 14px，核心数字 24-32px）。

交互： 典型的 Dashboard 交互，图表应具备 Hover 悬浮显示详细数据的功能。
