# 代码仓库设置页面 — 设计需求文档

本文档描述华为云 DevOps 平台「代码仓库设置」页面的详细设计需求，
用于指导 Pencil 设计工具基于组件库 `designsystem/devUI/components/devUI2.pen` 进行设计实现。

---

## 1. 整体布局与视觉风格

**布局结构**：经典多栏布局——顶部全局导航栏（全宽）+ 左侧工具链专用导航栏（窄栏，约64px宽图标导航展开态约200px）+ 左侧设置手风琴侧边栏（约200px宽）+ 中间主内容区（自适应剩余宽度）。在顶栏下方、主内容区上方有项目头信息区域（面包屑 + 项目信息 + 操作按钮）和仓库页签导航栏。

**整体结构示意**：
```
┌─────────────────────────────────────────────────────────┐
│                  A. 顶部全局导航栏（全宽）                   │
├────────┬────────────────────────────────────────────────┤
│        │      C. 项目头信息区（面包屑+项目信息+操作按钮）      │
│        ├────────────────────────────────────────────────┤
│   B.   │              D. 仓库页签导航栏（10个页签）          │
│ 工具链  ├─────────┬──────────────────────────────────────┤
│ 专用   │  E. 设置  │                                     │
│ 导航栏  │  侧边栏   │    F. 仓库设置主内容区（表单配置）       │
│        │ (手风琴)  │                                     │
│        │          │                                     │
└────────┴─────────┴──────────────────────────────────────┘
```

**视觉风格**：
- 色调：科技感蓝白系；页面背景浅灰 #F3F6F8～#EEF0F5；内容区白底 #FFFFFF
- 主色：蓝色 #526ECC（主操作按钮、选中态）、绿色按钮用于保存/提交
- 警告色：红/粉色 #FFEEED 背景 + #D4382D 文字用于告警提示
- 组件特征：直角/微圆角风格，表单输入框线框风格，按钮有圆角
- 文字：主文字 #252B3A、次文字 #575D6C、辅助文字 #8A8E99、链接文字蓝色 #526ECC、警告链接红色 #D4382D

**画布尺寸**：1920×1080
