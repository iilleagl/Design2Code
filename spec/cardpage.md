# 华为云控制台工作台页面 — Pencil 设计需求提示词

本文档为 Pencil 设计工具提供的详细需求描述，用于在参考组件库 `designsystem/devUI/components/devUI2.pen` 的前提下还原华为云控制台工作台（Dashboard）页面。请按「整体布局与视觉风格 → 核心区域详解 → 组件使用与还原要点」顺序理解并实现。

---

## 1. 整体布局与视觉风格

**布局结构**：采用经典「三明治」布局——顶部全局导航栏、左侧固定侧边导航、中间主内容区、右侧辅助信息栏。纵向为顶栏单行、主体左右分栏（左窄右宽，中间主内容区最宽），适合仪表盘/项目管理类概览页。

**视觉风格**：
- 整体色调为科技感蓝白系：背景为大面积浅灰色（可带微弱几何/代码符号底纹），卡片与操作区为纯白，主按钮与选中态使用蓝色（品牌色/交互色）。
- 组件特征：圆角矩形（卡片、按钮、输入框）、轻微阴影区分层级，留白充足，通过白底卡片和间距区分逻辑区块。
- 文字：主要文字深灰、次要/说明中灰、标签可用浅绿底+深灰字；图标为统一线条风格，辨识度高。

**空间利用**：顶栏固定高度；左侧栏固定宽度；主内容区与右侧栏比例约 3:1 或 4:1；卡片之间、区块之间留白充足，降低认知负担。

---

## 2. 核心区域详解

### A. 顶部全局导航栏

- **左侧**：
  - 汉堡菜单图标（三条横线）。
  - 「华为云」Logo 及品牌名。
  - 文本导航项依次：控制台、「华北-北京四」（带下拉箭头）、首页、工作台、效能洞察、服务（带下拉箭头）、华为开源镜像站。
  - 其中「工作台」为当前选中态（高亮）。
- **右侧**：
  - 通知图标（铃铛）。
  - 帮助图标（问号）。
  - 语言切换：显示「EN」「HZ」（中文），「HZ」为选中高亮。

**还原要点**：顶栏单行、全宽；左侧 Logo+导航链接、右侧图标+语言切换；下拉用下拉箭头示意，选中项需有视觉区分。

---

### B. 左侧侧边导航栏

- 背景纯白，固定于页面左侧，垂直列表。
- 标题：「我的关注」，旁有星形图标。
- 下方垂直导航项（每项左侧带图标）：
  - 默认项目（日历/文件夹图标）。
  - API服务创建的项目（API 图标）。
  - 已归档（文件夹/归档箱图标）。

**还原要点**：白底、竖排、图标+文字；可区分选中/悬停状态。

---

### C. 中间主内容区域

#### C.1 欢迎与最近访问区

- **欢迎语**：主内容区顶部大号文字「Hello, Jingwen」。
- **最近访问**：
  - 区块标题：「最近访问」。
  - 右侧为 3 张横向卡片，并排；每张卡片内含：彩色小图标 + 居中标题（示例：「Build持续测进个人项目规划说明」「DigitalCity智慧城市」「ROMA持续测进个人项目」）；卡片白底、圆角、轻阴影。
  - 该区域右侧可放置一张蓝色调 3D 插画（服务器机柜/数据流等科技感图形），与卡片并排或作为背景装饰。

**还原要点**：欢迎语字号突出；最近访问为「标题 + 多卡片横排 + 可选插画」结构。

#### C.2 所有项目区

- **标题**：「所有项目」，旁可有小圆点或装饰。
- **筛选与操作栏**（同一行或紧挨多行）：
  - **类型筛选**：水平标签/页签——「全部类型」（选中、蓝底）、项目群、IPD、Scrum、看板。
  - **操作**：「+ 新建」主按钮（蓝底、带下拉箭头）、「所有项目」下拉选择框、搜索框（占位符「请输入关键字搜索」、带放大镜图标）。
  - **视图切换**：「卡片」「列表」两档，当前为「卡片」选中高亮。
- **项目卡片列表**：
  - 三列网格，多行；每格一张项目卡片。
  - **单卡内容**：
    - 左上：彩色方形图标（示例：黄底黑字「M」）。
    - 右上：星形收藏图标 + 三点「更多」菜单。
    - 标题：如「MO安全编程规范」。
    - 标签：标题下两行浅绿底标签（如「标签」「标签」）。
    - 底部：小字「创建者: abc_abcdefghijklmn 项目ID: a2b...」。

**还原要点**：筛选栏与主按钮、下拉、搜索、视图切换同排；卡片网格 3 列；单卡含图标、标题、标签、元信息与操作图标。

---

### D. 右侧辅助信息栏

- 白底、圆角卡片式区块，与主内容区有视觉分隔（阴影/间距）。

- **公告**：
  - 标题「公告」。
  - 内容：一条品牌升级文案（示例：「【品牌升级】"软件开发平台DevCloud"及"项目管理ProjectMan"即日起正式更名"为软件开发生产线CodeArts"和"需求管理CodeArts Req"」）。
  - 文末「2天前」+ 省略号表示可展开。

- **活动**：
  - 标题「活动」。
  - 卡片内：主标题「频繁重新登录?」、副文案「点击查看如何增加登录保持时长 →」；可配蓝色调插画（如笔记本+显示器、登录/安全相关图形）。

- **帮助文档**：
  - 标题「帮助文档」，右侧「查看更多」链接。
  - 下方九宫格：每格为「图标 + 文本」链接，如：产品介绍、快速入门、用户指南、最佳实践、常见问题、计费说明、通用参考、用户论坛、版本说明。

**还原要点**：三块独立卡片（公告、活动、帮助文档）；帮助文档为图标+文字网格，约 3×3。

## 3. 组件使用与还原要点（参考 devUI2.pen）

实现时优先从 `designsystem/devUI/components/devUI2.pen` 中选用以下组件类型，并按需复制到设计稿后通过 ref 引用；引用后**必须**修改实例内的文案、图标、占位内容以符合本页业务语义。

| 区域 | 建议组件 | 说明 |
|------|----------|------|
| 顶栏 | Header / 顶部导航栏 | 支持 Logo、多链接、下拉、右侧图标与语言切换 |
| 顶栏下拉 | Dropdown / Select | 地区、服务、语言等下拉 |
| 左侧栏 | Sidebar / NavMenu | 垂直列表、图标+文字、选中态 |
| 各类卡片 | Card | 最近访问卡片、项目卡片、公告/活动/帮助文档区块；支持圆角、阴影、自定义内容（图标、标题、标签、描述、操作） |
| 主按钮 | Button（主色） | 如「+ 新建」 |
| 次按钮/链接 | Button（次要）/ 文本链接 | 如「查看更多」、视图切换未选态 |
| 搜索 | Input（带前置图标） | 占位符「请输入关键字搜索」、放大镜图标 |
| 筛选/页签 | Tabs / SegmentedControl | 全部类型/项目群/IPD/Scrum/看板；卡片/列表视图切换 |
| 下拉选择 | Select / Dropdown | 所有项目、地区等 |
| 图标 | Icon | 菜单、通知、帮助、收藏、更多、语言等，风格统一 |
| 排版 | Typography | 标题、正文、说明、时间戳，字号与色值符合设计系统 |

**设计规范**：颜色使用 Design Token（如品牌蓝、背景灰、主/次文字色）；字号建议 12/14/16/20/24/36 px；间距为 4 的倍数；圆角与阴影与组件库一致。

**还原优先级**：先搭好顶栏、左侧栏、主内容区与右侧栏的框架，再逐区块填入导航项、欢迎语、最近访问卡片、筛选栏、项目卡片网格、公告/活动/帮助文档；每使用 ref 的组件都需替换为上述文案与业务含义，避免保留组件默认占位内容。

---

## 4. 小结（供 Pencil 执行时自检）

- **布局**：顶栏 + 左栏 + 主内容（欢迎+最近访问+所有项目）+ 右栏（公告+活动+帮助文档）。
- **主内容**：欢迎语 → 最近访问（标题+3 卡片+插画）→ 所有项目（标题+筛选栏+卡片/列表切换+项目卡片网格）。
- **组件**：导航、卡片、按钮、输入、标签/页签、下拉、图标、排版均参考 devUI2.pen，复制组件后 ref 并必改文案与图标。
- **视觉**：蓝白灰主色、圆角与轻阴影、留白充足，便于还原原始页面的层次与信息结构。
